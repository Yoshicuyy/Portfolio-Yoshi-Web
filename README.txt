# Modern Dark Portfolio Website

A modern personal portfolio website template featuring an elegant dark mode design with a purple–black gradient background, smooth animations, and a fully responsive layout. Built using pure HTML, CSS, and JavaScript without any frameworks, making it lightweight, fast, and easy to customize.

This template is suitable for web developers, freelancers, UI/UX designers, and anyone looking to build a professional personal brand online.

Features:
- Modern, minimal, and premium design
- Dark mode by default with purple–black gradient background
- Fully responsive (mobile, tablet, and desktop)
- Hero section with circular profile photo and glow effect
- Sticky navbar with mobile menu toggle
- Smooth scrolling and scroll-based animations
- Skills section with programming language icons
- Contact section with social media icons
- Clean, well-structured, and beginner-friendly code
- No frameworks or heavy libraries required

Folder structure:
portfolio/
- index.html
- style.css
- script.js
- Foto/
  - img.jpg

How to use:
Open the `index.html` file and replace all placeholder content such as name, role, description, email address, social media links, and project details with your own information. All content is clearly written in plain HTML and easy to edit.

To change the profile photo, place your image inside the `Foto/` folder and name it `Logo.jpg`. Make sure the image path in `index.html` matches this location. For best results, use a half-body photo with a clean or dark background.

To customize the accent color, open the `style.css` file and edit the CSS variables inside the `:root` section. This allows you to quickly change the main color theme of the website.

Deployment:
You can easily deploy this website using GitHub Pages or Netlify. For GitHub Pages, upload all files to a repository and enable Pages from the main branch. For Netlify, simply drag and drop the project folder into the Netlify dashboard and the site will be live instantly.

Browser support:
This template works well on all modern browsers including Chrome, Firefox, Edge, and Safari.
